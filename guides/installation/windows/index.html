    Getting SplashKit installed on your system.

    1. Download the correct version of MSYS (32bit or 64bit depending on your computer).

    2. Run the installer

    3. Follow the installation wizard, with default values.

    4. Follow the installation steps on the MSYS2 page but executing the command pacman -Syu.

    5. Restart your terminal and relaunch MSYS2, then execute pacman -su

    6. In your MSYS2 Terminal, paste and run the following line: bash <(curl -s https://raw.githubusercontent.com/splashkit/skm/master/install-scripts/skm-install.sh). Found at splashkit.

    7. Restart your terminal, and test that SplashKit installed by executing skm in the MSYS2 terminal.

    8. go to the dotnet core page. Select command line then go to other downloads. Once there, select current and select the MacOS download.

    9. Run the installer and continue through the install wizard with default values.

    10. Download Visual Studio Code

    11. Run the installer.

    What’s next?

    Congratulations! If you’ve followed these steps correctly, then you will have installed all the tools needed for SIT771.

    Don’t worry if you run into any problems, just head over to the forum and we’ll be able to help you out!
