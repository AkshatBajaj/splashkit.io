    Install Dotnet Core Tools

    1. Install the dotnet core library, and follow the installation steps. Firstly, we need to install the Brew Package Manager.

    2. And execute it in the terminal

    3. Then copy each line of the install pre-requisites.

    4. Continue on executing the install pre-requisites on the dotnet core page by copy-pasting the commands.

    5. Install dotnet core SDK.

    6. Run the installer.

    7. To test if dotnet was installed successfully, execute the command: dotnet into your terminal.

      If you see command not found, don’t worry! Executing ln -s /usr/local/share/dotnet/dotnet /usr/local/bin should fix it.

    8. In Visual Studio Code you should install the following extensions:

    • C# (search for ms-vscode.csharp)
    • C# XML Documentation Comments (k--kato.docomment)

    • vscode-icons (robertohuertasm.vscode-icons)
